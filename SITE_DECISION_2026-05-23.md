# Site decision — Software / Tools affiliate site

Decision: build the first AI-generated affiliate site around software/tools/SaaS.

Working site concept: **Workflow Tools Review**  
Audience: solo operators, creators, remote workers, AI workers, small businesses.  
Language strategy: English-first for SEO scale; Traditional Chinese companion pages later when Taiwan intent is clear.  
Commercial model: comparison/review/tutorial content with affiliate links, disclosure, and non-affiliate alternatives.

## Why this site first

- Highest-scoring category in Affiliates.One analysis: `software_tools` avg score 58.2.
- Strong offers: Domain.com, MacPaw, WPS, Norton, NordVPN, PIA, AVG, Stellar, PicWish, DataCamp, AOMEI.
- Evergreen topics and high search intent.
- Works well with AI-generated content: comparison tables, checklists, FAQ, GEO direct answers, screenshots/source-backed claims.
- Does not require Kevin's existing technical brand positioning.

## First cluster

1. Domain/hosting: Domain.com, Network Solutions, eUKhost.
2. Mac/system maintenance: MacPaw, AVG, Norton.
3. Backup/recovery: AOMEI, Stellar, Tenorshare.
4. VPN/security: NordVPN, Private Internet Access, Surfshark, PureVPN, Ivacy, Zenmate.
5. Productivity/image tools: WPS, PicWish, CyberLink, Placeit, InVideo.
6. Skill-up: DataCamp, Preply, Mondly.
