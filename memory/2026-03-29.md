## 18:14 - Telegram 媒體回傳踩坑

### 問題
- 嘗試用 `MEDIA:./meijia_voice.mp3` 回傳音檔到 Telegram 失敗

### 真正原因
- OpenClaw 會把 `MEDIA:./...` 當成 **workspace 相對路徑**
- 當時實際檔案在 `/tmp/meijia_voice.mp3`
- 但系統嘗試讀的是 `/Users/kevin/.openclaw/workspace/meijia_voice.mp3`
- 因此出現：`LocalMediaAccessError: Local media file not found`

### 正確做法
- 若要用 `MEDIA:./...` 回傳媒體，必須先把檔案放進 workspace
- 例如：`/Users/kevin/.openclaw/workspace/out/mei-jia-10s.mp3`
- 再回：`MEDIA:./out/mei-jia-10s.mp3`

### 結論
- 不是 Kimi 不會發檔案
- 是 **MEDIA 相對路徑指錯位置**
