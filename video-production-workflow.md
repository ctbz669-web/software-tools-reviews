# 影片製作流程 - 草台班子研究室

## 標準流程

### 1. 腳本撰寫
- [ ] 確定主題和時長（3-5分鐘）
- [ ] 撰寫旁白稿
- [ ] 輸出：`epX-script.md`

### 2. 分鏡設計
- [ ] 依腳本拆分段落
- [ ] 設計每段畫面內容
- [ ] 標註字幕文字（**最多兩行**）
- [ ] 輸出：`epX-segments.md`

### 3. 圖片生成
- [ ] 使用 image_generate 生成素材
- [ ] 確保 16:9 比例（1920×1080）
- [ ] 檢查圖片完整性
- [ ] 命名：`epX_segment_01.png` ~ `epX_segment_08.png`

### 4. 配音生成
- [ ] **使用 macOS 本機語音 `meijai`**
- [ ] 測試語速（正常速度，不要太慢）
- [ ] 分段生成後合併
- [ ] 輸出：`epX-narration.mp3`
- [ ] **禁止使用 ElevenLabs**

### 5. 影片合成
- [ ] 時長嚴格匹配（圖片總時長 = 配音時長）
- [ ] 使用 ffmpeg 合成
- [ ] **字幕燒錄**（drawtext，白字黑邊）
- [ ] 添加 Logo「草台班子研究室」（右下角）
- [ ] 確保無黑邊
- [ ] 輸出：`epX-final.mp4`

### 6. 嚴格驗收
- [ ] 提取 8+ 個關鍵帧檢查
- [ ] 確認開頭無黑屏
- [ ] 確認字幕清晰、位置正確
- [ ] 確認 Logo 正確顯示
- [ ] 確認音畫同步
- [ ] 確認結尾正常

### 7. 交付
- [ ] 傳送給 Kevin 確認
- [ ] 記錄製作問題和改進點
