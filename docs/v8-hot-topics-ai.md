## 候選題材 1：AI Agent 企業化加速 - 從對話到執行

### 核心事實
- **2026年3月22日**：NVIDIA 推出 Agent Toolkit for Enterprise AI Agents，標誌 AI 從生成答案轉向執行實際工作
- **2026年3月24日**：Anthropic 讓 Claude 獲得電腦控制能力，可操縱滑鼠、鍵盤和螢幕完成任務
- **2026年3月19日**：Zoom 擴展 Agentic AI，將會議、通話和客戶體驗從洞察轉向執行
- **2026年3月18日**：Anthropic 推出 Dispatch 功能，讓用戶遠端在電腦上運行 AI Agent

### 數據亮點
- NVIDIA 企業 AI Agent 工具包支持大規模部署
- Zoom 的 AI 不再僅限於會議摘要，開始執行工作流程
- 多家企業級平台在同一周內發布 Agent 相關功能

### 適合影片角度
- AI Agent 從「聊天機器人」到「數位員工」的轉變
- 企業級 AI 應用進入執行階段

## 候選題材 2：OpenAI 策略調整 - Sora 關閉與模型小型化

### 核心事實
- **2026年3月25日**：OpenAI 關閉 Sora 視頻生成產品，引發對企業 AI 未來方向的討論
- **2026年3月20日**：OpenAI 推出 GPT-5.4 Mini 和 Nano，主打更快、更低成本的 AI 工作流程
- **2026年3月18日**：OpenAI 通過 AWS 擴展美國政府 AI 訪問，國防採用加速

### 數據亮點
- OpenAI 關閉其「視覺上最令人印象深刻的產品」
- GPT-5.4 Mini/Nano 針對開發者成本優化需求
- 政府 AI 訪問不再綁定單一供應商，改以雲端基礎設施擴展

### 適合影片角度
- OpenAI 從展示性技術轉向實用性產品
- AI 模型小型化趨勢：性能 vs 成本效益的平衡
