# 第 6 章：建立你的 AI 工具箱——上班族的工具組合

---

## 你需要的是「組合」不是「單一工具」

上一章說了不要只用一個 AI。但也不用太貪心，變成「工具收集狂」——帳號註冊了一堆，但每個都只用過一次。

你需要的是一套「實際會用到的組合」。

## 如何選擇工具：五個維度

選擇 AI 工具時，不要只看「功能」，還要考量現實因素：

| 維度 | 問題 |
|------|------|
| **取得難易度** | 要錢嗎？公司可以報帳嗎？ |
| **學習曲線** | 我能在一天內學會基本操作嗎？ |
| **與現有工具整合** | 能串接到我平常用的軟體嗎？ |
| **輸出品質** | 這個工具的輸出品質穩定嗎？ |
| **公司政策** | 公司允許使用這個工具嗎？有沒有合規要求？ |

### 常見的現實考量

```
💰 成本考量
- 免費版能用嗎？付費版值得嗎？
- 有沒有公司帳號可以申請？

🔒 合規考量
- 金融業、醫療業有特殊規範
- 公司資料能不能上傳到外部 AI？

🔧 串接考量
- 能用 API 串接到現有流程嗎？
- 能在既有的軟體（Slack、Notion）裡使用嗎？
```

## 各類工具詳細分析

### 文字 AI：ChatGPT vs Claude

| 維度 | ChatGPT | Claude |
|------|---------|--------|
| **強項** | 創意發想、翻譯、代碼 | 長文本分析、深度思考 |
| **弱項** | 有時過度正面、取悅用戶 | 創意發想稍弱 |
| **適合場景** | 快速郵件、日常寫作 | 重要報告、策略分析 |
| **費用** | 免費/Plus $20 | 免費/Pro $20 |
| **公司版** | ChatGPT Team | Claude Team |

#### 實際選擇建議

```
情境 1：你要寫一篇即時的行銷文案
→ 選 ChatGPT，創意發想速度快

情境 2：你要分析一份 50 頁的年度報告
→ 選 Claude，長文本處理更穩定

情境 3：你要翻譯一篇學術論文
→ 兩者皆可，但 Claude 在學術英語上更精準

情境 4：你要做複雜的策略分析
→ 選 Claude，邏輯推理能力更強
```

### 搜尋 AI：Perplexity vs Copilot

| 維度 | Perplexity | Copilot |
|------|------------|---------|
| **強項** | 即時資訊、引用來源 | 與 Microsoft 365 整合 |
| **弱項** | 深度分析較弱 | 需要 Microsoft 生態系 |
| **適合場景** | 快速研究、事實查證 | 工作流程已在 Microsoft 環境 |
| **費用** | 免費/Pro $20 | 免費 |

#### 實際選擇建議

```
情境 1：你要研究一個新的技術主題
→ 選 Perplexity，會標明來源，方便查證

情境 2：你要總結一篇網路文章的重點
→ 兩者皆可

情境 3：你已經在用 Bing 和 Microsoft 365
→ 選 Copilot，與現有工具整合更好

情境 4：你要找最新的產業趨勢
→ 選 Perplexity，更即時
```

## 常見的錯誤：工具收集癖

這是一個常見的陷阱：

```
症狀：
- 帳號註冊了 15 個 AI 工具
- 每個工具都打開看過一次
- 實際在用的只有 2 個

問題：
不是「工具不夠用」，是「沒有建立實際在用的組合」

解決：
選擇 2-3 個最實用的，學到極致
```

```
工具名稱    主要用途    使用頻率    滿意度
─────────────────────────────────────────
ChatGPT     ?          ?/10        ?/10
Claude      ?          ?/10        ?/10
其他...     ?          ?/10        ?/10
```

找出：
- 使用頻率高但滿意度低的 → 考慮更換

- 缺少的功能 → 找一個新工具補充

## 工具組合的實際案例

### 案例一：普通上班族的日常組合

```
早晨：
1. Perplexity → 查看今日產業新聞（15 分鐘）
2. ChatGPT → 起草今日待辦事項（5 分鐘）
3. DeepL → 翻譯一封英文郵件（2 分鐘）

午後：
4. ChatGPT → 潤稿商業郵件（10 分鐘）
5. Claude → 分析會議記錄（15 分鐘）

下班前：
6. ChatGPT → 生成明日會議大綱（10 分鐘）
```

### 案例二：行銷人員的內容組合

```
內容創作流程：
1. ChatGPT → 發想本週的 10 個內容主題
2. Claude → 為每個主題設計大綱
3. Midjourney → 為每個主題生成配圖
4. Canva AI → 排版設計
5. ChatGPT → 生成社群文案
6. Claude → 最終校對和調整語氣
```

## 常見問題

### Q：免費版和付費版差很多嗎？

```
一般任務：差別不大
免費版缺點：
- 有用量限制
- 回應速度較慢
- 沒有最新模型

建議：
- 先用免費版測試工具適合不適合你
- 覺得工具值得，再升級付費版
```

### Q：需要一次擁有所有工具嗎？

```
不需要。
建議流程：
Step 1：選擇 1-2 個核心工具用到熟練
Step 2：發現需求缺口，再找對應工具
Step 3：測試新工具 1-2 週
Step 4：如果真的提升效率，再加入工具箱
```

### Q：國外的工具在台灣能用嗎？

```
大部分可以，但有些例外：
- 部分工具對中國境內有限制
- 部分工具需要境外電話驗證
- 信用卡付款可能有限制

建議：準備一張境外可用的信用卡
```

## 本章重點

1. **不需要過度工具化**，選擇「實際會用」的組合
2. **五個選擇維度**：取得難易度 / 學習曲線 / 整合度 / 輸出品質 / 公司政策
3. **避免工具收集癖**：2-3 個用熟的，勝過 15 個只用一次的
4. **根據工作類型**（內容/研究/管理）擴展工具箱
5. **工具組合案例**：普通上班族 vs 行銷人員
