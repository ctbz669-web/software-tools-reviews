## 🚀 使用方式

### 方式一：瀏覽器直接開啟
1. 開啟 `index.html`
2. 使用鍵盤方向鍵切換投影片
3. 按 `F` 進入全螢幕

### 方式二：匯入 Google Slides
1. 在瀏覽器開啟 `index.html`
2. 複製所有內容
3. 貼到 Google Slides（建議用每頁一張的方式重建）

### 方式三：PDF 輸出
1. 開啟 `index.html`
2. 在網址列前面加 `?print-pdf`
3. 按 `Ctrl+P` (Mac: `Cmd+P`) 列印成 PDF
