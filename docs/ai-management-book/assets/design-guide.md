# 《AI 管理學》視覺設計指引

---

## 封面設計概念

### 概念一：管理儀表板風格

**畫面構成**：
- 中央：一個正在運作的齒輪或管理儀表板圖示
- 主題：象徵「系統化管理」
- 配色：科技藍 + 組織橙，傳達「科技 + 效率」

**文案配置**：
```
主標題：AI 管理學
副標題：用管理框架駕馭 AI 能力
作者：草台班子
```

---

### 概念二：人類+AI 協作風格

**畫面構成**：
- 左側：人的大腦圖示（手繪風格）
- 右側：電路板式的 AI 網絡
- 中央：兩者連接的火花或連結線

**配色**：深灰 + 亮綠（象徵成長 + 連接）

---

### 概念三：北歐簡約風

**畫面構成**：
- 大量留白
- 簡單的線條圖：OKR、齒輪、流程圖
- 文字為主，視覺點綴

**配色**：黑白 + 一個強調色（如：Google Blue #4285F4）

---

## 內頁排版建議

### 紙張尺寸
- PDF/EPUB：A5（14.8 x 21 cm）——適合行動閱讀

### 字體建議

| 用途 | 字體建議 |
|------|----------|
| 標題 | Noto Sans TC Bold 或 PingFang TC |
| 正文 | Noto Sans TC Regular 或系統預設 |
| 強調 | 粗體或底線 |
| 引言 | 斜體 + 左側灰線 |

### 行距與留白
- 正文行距：1.8
- 段落間距：1.5 行
- 頁面留白：上下 2cm，左右 2.5cm

### 重點標示
```
【框架名稱】用粗體 + 底色標示
| 表格 | 灰底白字，交替列
💡 反思問題 | 淡黃底
✅ 行動清單 | 淡綠底 + 勾選方塊
```
