# AI 管理學快速參考卡

---

## 給 AI 下指令

```
✓ 給情境、給目標、給格式要求、給禁止事項
✗ 模糊指令（「幫我做...」「好不好...」）
✗ 不給角色或背景
```

---

## AI 輸出品質檢查四維度

| 維度 | 問題 |
|------|------|
| 準確性 | 事實是對的嗎？數字有驗證嗎？ |
| 相關性 | 回答跟問題相關嗎？ |
| 完整性 | 有遺漏重要資訊嗎？ |
| 可執行性 | 建議能直接執行嗎？ |

## PDCA 持續改進

```
Plan → 我要給 AI 什麼指令？
  ↓
Do → AI 執行並回答
  ↓
Check → 四維度檢查
  ↓
Act → 修正指令 or 接受結果
  ↓
   ↺ Loop
```

---

## Prompt 模板關鍵要素

1. **C (Context)** 背景脈絡
2. **L (Length)** 長度要求
3. **E (Examples)** 提供範例
4. **A (Audience)** 目標讀者
5. **R (Role)** 賦予角色

---

## AI 風險速查

| 風險 | 應對 |
|------|------|
| 資料外洩 | 不貼機密文件 |
| AI 幻覺 | 數字驗證、要求標明來源 |
| 過度依賴 | 人類保留最終決策 |

---

## 三階段成長路徑

| 階段 | 特徵 |
|------|------|
| 新鮮人 | 模糊指令、全盘接受或否定 |
| 上手者 | 精準指令、建立模板、審核習慣 |
| 高手 | 多工具協作、持續改進、團隊分享 |
