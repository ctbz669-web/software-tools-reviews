# Prompt 模板庫

---

## 模板 001：商業報告生成

```
【情境】
[你是誰] + [需要做什麼報告]

【目標】
[報告的目的]

【讀者】
[這份報告的讀者是誰？]

【要求】
- 長度：[具體字數或頁數]
- 風格：[專業/輕鬆/簡報風]
- 核心訊息：[最希望讀者記住的一件事]

【格式】
[大綱或特定格式要求]

【禁止】
[特別不想出現的內容]
```

## 模板 002：郵件撰寫

```
【情境】
我是[你的職位/角色]

【郵件類型】
[詢問/請求/道歉/感謝/跟進]

【對象】
[收件人是誰？]

【主要訊息】
[這封信最重要的訊息是什麼]

【語氣】
[正式/友好/堅定/誠懇]

【長度】
[不超過X字]

【特別要求】
[需要的行動：請對方確認/回覆/批准]
```

## 模板 003：創意發想

```
【主題】
[你想要發想的內容]

【目標】
[想要達到的目的]

【數量要求】
[想要X個點子]

【角度要求】
[如：前5個實用導向/後5個創意導向]

【限制條件】
[X個字以內/符合某種風格/針對某族群]

【目標讀者】
[這是給誰看的]
```

## 模板 004：翻譯潤稿

```
【原文】
[在這裡貼上原文]

【語言方向】
[中→英 / 英→中 / 其他]

【用途】
[學術論文/商業文件/行銷素材/個人信件]

【風格要求】
[正式/學術/輕鬆/創意/白話]

【特別要求】
[專有名詞處理/保留原意/在地化]

【目標讀者】
[這份譯文的讀者是誰]
```

## 模板 005：SWOT 分析

```
請用 SWOT 分析框架分析 [主題/公司/產品]：

【S - 優勢】
- Strengths（內部優勢）：[]
- 
【W - 劣勢】  
- Weaknesses（內部劣勢）：[]

【O - 機會】
- Opportunities（外部機會）：[]

【T - 威脅】
- Threats（外部威脅）：[]

【額外要求】
[針對這個 SWOT 的策略建議？/對應每個項目的具體行動？]
```

## 模板 007：自我審核 Prompt

```
請你扮演一個專業審核者，從以下維度評估你的回答：

1. 準確性：有事實需要驗證嗎？
2. 相關性：有答非所問的部分嗎？
3. 完整性：有遺漏的重要觀點嗎？
4. 可執行性：建議的步驟具體嗎？

如果有任何問題，請先修正後再給我最終版本。
```
